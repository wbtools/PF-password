# PF-password

一个用 Python 编写的简单密码管理工具。

## 描述

这个工具允许用户使用本地 SQLite 数据库安全地存储和管理他们的密码。它提供了添加、检索和删除密码的基本功能。

## 功能

- 添加新密码
- 检索现有密码
- 删除密码
- 使用 SQLite 数据库进行安全存储

## 安装

1. 克隆仓库：
   ```bash
   git clone https://github.com/yourusername/PF-password.git
   ```
2. 进入项目目录：
   ```bash
   cd PF-password
   ```
3. 安装所需依赖：
   ```bash
   pip install -r requirements.txt
   ```

## 使用方法

要运行密码管理器，请执行以下命令：

```bash
python pass.py
```

按照屏幕上的提示来管理你的密码。
